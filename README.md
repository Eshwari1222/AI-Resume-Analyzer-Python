A powerful Full-Stack Python application that uses Natural Language Processing (NLP) to analyze PDF resumes and match them against specific job roles.

## üåü Key Features
- **PDF Text Extraction**: Seamlessly reads and parses uploaded PDF resumes.
- **AI Matching Engine**: Uses **TF-IDF Vectorization** and **Cosine Similarity** to calculate a matching percentage between the resume and job requirements.
- **Skill Gap Analysis**: Identifies missing skills based on the selected job role (e.g., Data Scientist, Web Developer, AI Engineer, or Full Stack Developer).
- **Dynamic Results**: Provides a visual match score and a matching verdict (Excellent, Good, or Needs Improvement).
